## Uruchomienie:

Zainstaluj node.js
Będąc w folderze z projektem uruchom komendę: 
```bash
npm install
```

Aby uruchomić lokalny serwer pod adresem http://localhost:3000 użyj komendy: 
```bash
node server.js
```
